# Online Appendix

This repository hosts the online appendix for the article: **"Voting Against the Giants: Legislative Performance and Minor Party Survival under Taiwan’s Mixed-Member System"** by Ya-Han Chen and Yung-Ming Hsu.

This appendix provides additional analyses and pretreatment covariate balance checks supporting the main findings.

The appendix is available in this repository as:
- `Appendix_Chen and Hsu.pdf`

# Citation
Ya-Han Chen and Yung-Ming Hsu. Forthcoming. **"Voting Against the Giants: Legislative Performance and Minor Party Survival under Taiwan’s Mixed-Member System."** *Issues and Studies*.
